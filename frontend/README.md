# AI Video Ad Creator

A powerful SaaS platform for creating stunning video ads for beauty products, fashion, and lifestyle brands.

## Project Overview

**AI Video Ad Creator** is a comprehensive video creation platform that enables brands to create professional video content without the need for expensive agencies or complex editing software.

## Key Features

- **500+ Professional Templates** - Industry-specific templates for beauty, fashion, and lifestyle
- **AI-Powered Editing** - Automatic color correction, product highlighting, and caption generation
- **Multi-Platform Export** - Optimized for Instagram, TikTok, YouTube, Facebook, and more
- **Brand Kit Management** - Save and reuse brand colors, fonts, and logos
- **Collaboration Tools** - Team workflows and client approval systems
- **Analytics & Performance** - Track engagement and conversion metrics

## Target Audience

- Beauty Brands
- Fashion Labels  
- E-commerce Stores
- Influencers
- Marketing Agencies
- Small Businesses

## Technologies Used

This project is built with modern web technologies:

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Build Tool**: Vite
- **State Management**: React Context + TanStack Query
- **Routing**: React Router v6
- **Animations**: Custom animation hooks + Framer Motion

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd ai-video-ad-creator

# Install dependencies
npm install

# Start development server
npm run dev
```

### Development

```sh
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linting
npm run lint
```

## Deployment

The project can be deployed to any static hosting platform:
